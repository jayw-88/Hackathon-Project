# AI Fossil Scanner
An AI App that allows one to upload an image of a fossil and have AI detect what the fossil could possibly be.

Created by JW and VM.

### How to Run the Application
This app can be run on Streamlit by accessing this link: https://ai-fossil-detection.streamlit.app/

## Disclaimer:
AI can make mistakes. The AI is being trained frequently, so each time it is more accurate in detecting fossils.
