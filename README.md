# AI Fossil Scanner
An AI application that allows one to upload an image of a fossil and have AI detect what the fossil could possibly be.

Created by Jayden Wu and Viraj Malhotra.

### How to Run the Application
This app can be run on Hugging Face Spaces by accessing this link:
https://huggingface.co/spaces/jayw88/Fossil-Scanner-AI

### Disclaimer:
The AI can make mistakes and detect inaccurate fossils.
